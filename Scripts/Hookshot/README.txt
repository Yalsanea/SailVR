This folder contains the scripts that are related to the hookshot/grappling hook mechanic of the game. A description of each script is show below.


Shoot.cs
This script contains the main controls of the hookshot. It is attached to the player object of the game.

HookCollider.cs
This script contains the max range as well as the pull speed of the hookshot. It is attached to the hook prefab.

HookBlocks.cs
This script allows the object to interact with the hookshot. It is attached to the game objects that the player hooks to.
